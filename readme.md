This replicates the remember/know experiment using Tulving's humorous word-definition pairs used in Soderlund(2008) and Palombo (2015). 72 word-definition pairs are shown or auditorily
presented to the participants across a total of 4 blocks . Each block contains 18 such pairs . The first and third block present the stimuli visually . The second and fourth block 
presents that stimuli auditorily. There are 2 delays for retrieval - 1. 30 mins ; 2. 24 hours . During retrieval , the following is tested -

1.Cued Recall: The screen shows the definition and a blank text box. The user types the word.

2.Recognition: The screen asks, "Did you see this definition earlier? (Y/N)".

3.Source Memory: If they press Yes, it asks, "Was it Auditory or Visual?".

4.Remember/Know: It asks them to categorize their memory (Remember, Know, or Guess).
